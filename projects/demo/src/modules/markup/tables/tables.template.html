Use .tui-table class to add Taiga styles to a table
An advanced separate @taiga-ui/table package with interactive tables is here

Elements

  • .tui-table__tr
  • .tui-table__th
  • .tui-table__td
  • .tui-table__subtext : small secondary text into a cell
  • .tui-table__sort : sorting action button
  • .tui-table__sort-icon : sorting action button icon

Modifiers

  • .tui-table_layout_fixed : fixed cell width
  • .tui-table_font-size_l : "l" text size for the whole table
  • .tui-table_font-size_s : "s" text size for the whole table
  • .tui-table_size_l : "l" cell size for the whole table
  • .tui-table__tr_border_top : border top for the row
  • .tui-table__tr_border_none : removes border bottom
  • .tui-table__tr_hover_disabled : removes hovered state of row
  • .tui-table__tr_cursor_pointer : adds "cursor: pointer" for hovered row
  • .tui-table__th_font-size_l : "l" text size for table headings
  • .tui-table__td_font-size_l : "l" text size for a single cell
  • .tui-table__td_font-size_s : "s" text size for a single cell
  • .tui-table__td_size_l : "l" size for a single cell
  • .tui-table__td_align_center : align center for a single cell
  • .tui-table__td_first, .tui-table__th_first : removes left padding of the cell
  • .tui-table__td_last, .tui-table__th_last : removes right padding of the cell
  • .tui-table__td_text_center, .tui-table__th_text_center : alignes text center in a cell
  • .tui-table__td_text_right, .tui-table__th_text_right : alignes text right in a cell
  • .tui-table__td_text_overflow : adds "..." when text overflows
  • .tui-table__sort_active : active state for sort button
  • .tui-table__sort_up : arrow up state for sort button
