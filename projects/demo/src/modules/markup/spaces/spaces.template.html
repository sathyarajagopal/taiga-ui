You can use global classes or LESS mixins to make (n multiples 4px) margins
Classes are included in global styles of core library, you do not need to setup them
You can build a class with direction and value between 0 and 15 ( tui-space_<direction>-<value> ).
Mixins are included in local LESS styles, so you need to import them into LESS styles file and add it to classes of your component

Mixin also gets a direction and a value ( .tui-space(<direction>, <value>); ).
