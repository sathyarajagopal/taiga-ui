Class tui-container makes a container inside content
Class tui-row and set of tui-col_* classes make a grid inside a container

Container classes

  • .tui-container : width 1104px on screens >= 1280px and 824px on screens <= 1279px
  • .tui-container_adaptive : width 1104px on screens >= 1280px, 824px on screens >= 1024px and <= 768px, 100% on screens < 768px
  • .tui-container_fullwidth : width 100% with 24px margins on screens >= 768px and 16px on screens <= 767px
  • .tui-container_fixed : width 1104px

Columns classes for adaptive layout

  • .tui-col_lg-* : screen width >= 1280px (Desktop M)
  • .tui-col_md-* : 768px =< screen width <= 1279px (Desktop S)
  • .tui-col_xs-* : 360px =< screen width <= 767px (Mobile)
